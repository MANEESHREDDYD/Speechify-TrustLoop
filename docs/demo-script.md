# Demo Script

## Opening

Voice-first AI products are expanding across learning, meetings, podcasts, and work agents. As generated outputs become more important, users need to know whether each output is accurate and grounded.

S TrustLoop explores that reliability challenge as an independent, local-first prototype.

## Demo 1: Student Learning

1. Open Student Demo.
2. Load Voice AI and Accessible Learning.
3. Generate a summary and inspect its Trust Score.
4. Generate a quiz and open learning memory.

Key message: S TrustLoop turns passive listening into measurable learning.

## Demo 2: Meeting Notes

1. Open the product launch transcript.
2. Generate meeting notes.
3. Inspect action items, decisions, and claim-level evidence.
4. Evaluate the seeded wrong notes and show contradiction detection.

Key message: S TrustLoop makes meeting notes auditable.

## Demo 3: AI Work Report

1. Open the market analysis source pack.
2. Generate a work report.
3. Show grounded findings, missing topics, and unsupported claims.
4. Open the analytics dashboard.

Key message: S TrustLoop is a quality layer for AI agents that create finished work.
