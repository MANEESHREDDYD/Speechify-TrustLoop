# Demo Script

## Opening

Voice-first AI products are expanding across learning, meetings, podcasts, and work agents. This demo explores how generated outputs could be made easier to review against source text.

S TrustLoop is an independent, local-first portfolio prototype. It uses deterministic generation, lexical matching, and simple rules; it does not prove factual correctness.

## Demo 1: Student Learning

1. Open Student Demo.
2. Load Voice AI and Accessible Learning.
3. Generate a summary and inspect its heuristic review score.
4. Generate a quiz and open the demo topic signals.

Key message: S TrustLoop makes source overlap and skipped topics visible. It does not measure learning.

## Demo 2: Meeting Notes

1. Open the product launch transcript.
2. Generate meeting notes.
3. Inspect action items, decisions, and retrieved source evidence.
4. Evaluate the seeded wrong notes and show the limited date, owner, and requirement rules.

Key message: S TrustLoop demonstrates an inspectable review workflow for meeting notes.

## Demo 3: AI Work Report

1. Open the market analysis source pack.
2. Generate a work report.
3. Show grounded findings, missing topics, and unsupported claims.
4. Open the analytics dashboard.

Key message: S TrustLoop demonstrates a review layer for generated work.

The demo data and negative tests are synthetic and curated. They demonstrate stable product behavior, not generalization.
